¿Qué es?

LadderVoice

LadderVoice es una herramienta web para programar PLCs en lenguaje Ladder usando la voz como interfaz principal. Un agente de IA interpreta los comandos hablados y los traduce a instrucciones de control lógico que pueden compilarse y cargarse al PLC.

La herramienta incluye un editor visual de rungs, un panel de monitoreo de I/O, terminal de compilación y un asistente conversacional que entiende el programa activo.

Contexto

El problema

La programación de PLCs industriales requiere software especializado de escritorio (TIA Portal, CX-Programmer), interfaces complejas y un aprendizaje significativo antes de poder crear programas funcionales.

En entornos industriales, la necesidad de consultar manuales, alternar entre pantallas y escribir cada contacto manualmente reduce la eficiencia y aumenta el riesgo de error humano.

Propuesta

La solución

LadderVoice elimina la fricción: el técnico dicta en lenguaje natural ("agrega un contacto NO en I0.0 para el paro de emergencia") y el agente construye el rung correspondiente en el editor.

La IA recibe el estado del programa activo, las variables definidas y los rungs existentes — no responde de forma genérica, sino en contexto con tu lógica de control actual.

Flujo

¿Cómo funciona?

Usuario dicta un comando de control
Web Speech API transcribe la voz
LLM interpreta la instrucción con contexto del programa
Se genera el rung o acción ladder
Se compila y carga al PLC por Modbus

Capacidades del sistema

Control por voz

Dicta instrucciones en español. La IA entiende el contexto del programa y responde en consecuencia.

Editor Ladder visual

Edita rungs con contactos, bobinas, timers y bloques funcionales en una interfaz tipo IDE.

Terminal de compilación

Logs en tiempo real de compilación, carga y ejecución del programa en el PLC.

Monitor de I/O

Estado en vivo de entradas, salidas y marcas del PLC con tabla de watch y forzado.

Agente contextual

La IA conoce el programa activo: variables, rungs y estados — no responde de forma genérica.

Respuesta por audio

El agente puede responder por voz (TTS) para mantener las manos libres del operador.

Stack

Tecnologías utilizadas

Web Speech API

Reconocimiento de voz nativo del navegador. Sin instalación, sin backend adicional.

Vanilla ES6

Sin frameworks. Módulos nativos, estado simple, rendimiento máximo.

LLM (Groq / Ollama)

Agente de lenguaje natural para interpretar comandos de control industrial.

Modbus TCP

Protocolo industrial para comunicación con PLCs en red local.

GitHub Pages

Frontend estático desplegado. Sin instalación para la interfaz web.
