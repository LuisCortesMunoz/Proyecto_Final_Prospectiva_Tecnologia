Proyecto Final en Desarrollo

Agente de Voz para
Control de Automatización

Este repositorio documenta los pasos para el Proyecto Final, el cual trata de generar un agente capaz de, mediante comandos de voz, activar y desactivar diferentes elementos de automatización — ya sea encender una luz, mover una banda, controlar un UR3, y más.

Ver Documentación Repositorio
¿De qué trata?

Sistema de Control por Voz
para Ambientes Industriales

Reconocimiento de Voz

Captura y procesa comandos de voz en tiempo real usando modelos de lenguaje natural para interpretar instrucciones.

Agente Inteligente

Un agente central toma decisiones y enruta los comandos a los elementos de automatización correctos sin intervención manual.

Control Multidispositivo

Interfaces hacia PLCs, robots UR3, bandas transportadoras, actuadores de luz y más dispositivos industriales.
